📘 What is NeurAloud?

NeurAloud is an advanced document reader that turns PDFs, DOCX, and text into speech, with full highlighting and voice customization.

🧠 Features

  • PDF, DOCX, TXT, EPUB support
  • 100-file Library + 10-file Playlist
  • Auto-resume, sentence highlighting
  • 1+ hours free reading daily

🎧 Usage

  1. Upload a file
  2. Select engine + voice
  3. Click ▶️ to listen
  4. Save to Library/Playlist

🕒 Benefits

  • Multitask while listening
  • Visually impaired friendly
  • Supports 100+ voices and languages
